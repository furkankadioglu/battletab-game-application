# BattleTab Game Application

## Proje Hakkında
Multiplayer bir savaş oyunu uygulaması.

## Geliştirme Kuralları
- Her değişiklikten sonra açıklayıcı bir commit oluştur ve `git push origin main` ile push yap.
- Her yeni dosya veya önemli değişiklikte README.md'yi güncelle.
- Kod yazdıktan sonra mutlaka test et.
- Değişiklikleri `claude_documentations/` klasöründe belgele.

## Proje Yapısı
```
battletab-game-application/
├── .claude/                   # Claude Code proje konfigürasyonu
│   ├── CLAUDE.md              # Proje talimatları
│   └── settings.json          # Proje ayarları
├── README.md                  # Proje açıklaması
└── claude_documentations/     # Geliştirme dokümantasyonları
```
